# 手机号前三后四查询

## 工具介绍

在线手机号前三后四中间四位查询工具，可以根据公开的手机号段分配规则，根据归属地区域快速推算出被遮掩的号码中所有可能的号码。

## 如何使用

选择手机前3位号段，根据号码实际归属地情况选择该号码的归属地省份和城市，填写手机号最后4位，可以填写0-4位，不足4位将会自动前置补全。

归属地选择和后四位的填写越精确，可以排除的号码越多，筛选后的号码越少，查找效率越高。
